Gesture-Controlled Arduino Car

Problem Statement: To design a car that can be controlled using hand gestures, enhancing accessibility and promoting innovation in robotic systems.

Project Objective: To develop a wireless, touchless car controlled by hand movements via sensors, showcasing intuitive human-robot interaction and encouraging technological exploration in robotics and embedded systems.

Real-World Problems Addressed:

  • Hands-free control for individuals with limited mobility
  • Safer interaction in hazardous environments
  • Affordable access to robotics education

Key Components: Arduino Uno, Arduino Nano, nRF24L01, MPU6050 Sensor, L298N Motor Driver, Custom PCB, TT Gear Motors, 18650 Batteries.

Future Enhancements: Solar-powered systems, mobile application integration, advanced battery optimization.

Room Entry Counter

A smart system developed using basic integrated circuits (ICs) and sensors to count the number of people entering or exiting a room. Using infrared (IR) sensors at the entrance, the system updates the count on a 7-segment display or LCD, providing an efficient solution for monitoring room occupancy automatically.

Applications: Libraries, classrooms, offices, hospitals, and any other environment where people flow monitoring is important.

Key Features:

  • Automatic bidirectional counting
  • Real-time display updates
  • Capacity alerts for safety compliance

This project reflects my strong command of digital electronics, logical thinking, and practical implementation of sensor-based systems.
